Projects

Particle Life: An Artificial Life System

Particle Life is a final year bachelor project that explores the dynamics of artificial life through a force-based system with incorporated evolutionary mechanisms. Developed using Processing, this project delves into simulating emergent behaviors akin to life forms within a digital environment.

Merre Librin

MerreLibrin, is a platform I developed to help people to donate school books to people that need them.

Store POS

This is a desktop application made to be used locally for grocery stores, this app helps on calculating the amount for each sale records all the sales, helps keep the store stocked where the manager can easily see and edit any product in the store. I made this application using electron and it uses sqlite for the database. If you need such an application or know someone who does or similar apps feel free to message me.

Symbo Search

A google extension that integrates openai into every single google search you do, instead of being the google killer maybe OpenAi can live symbiotically with google search. I created this google extension to help people who want to get some quick answers on specific topic especially helping with code snippets, even though it works with any kind of query that OpenAi can answer since the answer come directly from it.
